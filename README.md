# ascentxai
AscentX.ai is a career-progression intelligence tool built with TypeScript. It analyzes the "delta" between your current professional footprint (CV, GitHub, LinkedIn) and your career goals to provide a strategic roadmap for leveling up.

## 🚧 Under Development
AscentX.ai is currently in active development and in an early development phase. The basic functionality — resume information extraction and GitHub profile retrieving — is already working, but many planned features are still in progress. Its primary goal is to provide hands-on experience with Agentic AI using frameworks like Mantra and Vercel.
